# Bugcrowd VRT — Interactive Mind Map

Interactive visual explorer for Bugcrowd's Vulnerability Rating Taxonomy (VRT). Browse 18 categories and 100+ vulnerability types with severity ratings, detailed descriptions, and real-world examples.

**[Live Demo](https://roshanxcvi.github.io/bugcrowd-vrt/)** · Developed by **[roshanxcvi](https://github.com/roshanxcvi)**

---

## Features

- Interactive mind map with expandable branches
- Detailed info panel on every node — how the bug works, impact, and examples
- Color-coded priority ratings (P1 Critical → P5 Info)
- Search by name or description
- Dark / Light mode toggle
- Pan, zoom, and touch support
- Single HTML file — no build step, no dependencies

## Categories Covered

Server-Side Injection · XSS · Broken Authentication · Sensitive Data Exposure · Server Misconfig · IDOR · CSRF · Unvalidated Redirects · Insufficient Security Config · Broken Access Control · Application-Level DoS · Cryptographic Issues · Mobile Security · AI/ML Security · Blockchain/dApp · Indicators of Compromise · Automotive Security · Client-Side Injection

## Data Source

Based on [Bugcrowd's open-source VRT](https://github.com/bugcrowd/vulnerability-rating-taxonomy) (Apache 2.0).
